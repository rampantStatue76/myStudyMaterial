## 使用浮动
例子1：（使用浮动+负边距）
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <div class="container">
        <div class="center">中间</div>
        <div class="left">左边</div>
        <div class="right">右边</div>
    </div>

    <style>
        * {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }
        .center {
            background-color: green;
        }
        .left {
            width: 200px;
            background-color: red;
        }
        .right {
            width: 200px;
            background-color: blue;
        }

        .center,.left,.right {
            float: left;
        }
        .container {
            overflow: hidden;
            padding: 0 200px;
        }
        .center {
            width: 100%;
        }
        .left {
            margin-left: -100%;
            position: relative;
            left: -200px

        }
        .right {
            margin-right: -200px;
        }
    </style>
</body>
</html>
```

例子2：（使用flex布局）
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <div class="container">
        <div class="center">中间</div>
        <div class="left">左边</div>
        <div class="right">右边</div>
    </div>

    <style>
        * {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }
        .center {
            background-color: green;
        }
        .left {
            width: 200px;
            background-color: red;
        }
        .right {
            width: 200px;
            background-color: blue;
        }

        .container {
            display: flex;
        }
        .center {
            flex: 1
        }
        .left {
            order: -1
        }
    </style>
</body>
</html>
```
